This project demonstrates the integration of **Snort IDS** with **Splunk SIEM** to simulate a real-world Security Operations Center (SOC) environment.

## Overview
- Configured **Snort** as a Network Intrusion Detection System (NIDS)
- Created custom Snort rules to detect:
  - Port scanning (Nmap)
  - ICMP / DoS attacks
  - FTP and SSH access attempts
- Forwarded Snort alerts to **Splunk Enterprise** using **Splunk Universal Forwarder**
- Analyzed and visualized security events using Splunk dashboards

## Tools & Technologies
- Snort IDS
- Splunk Enterprise
- Splunk Universal Forwarder
- Linux (Ubuntu)
- Nmap
